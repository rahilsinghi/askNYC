## Design Context

### Users
NYC residents, tourists, and hackathon judges. They interact via voice + camera on a phone, while a dashboard displays cinematic data visualizations on a separate screen. The job: point at a NYC location, speak a question, and instantly see structured civic data (health grades, safety incidents, building permits) rendered as an intelligence briefing.

### Brand Personality
Cinematic, Intelligence, Urban. The interface should feel like a high-tech city command center -- confident, precise, and atmospheric. Think Bloomberg Terminal meets a Christopher Nolan film's control room.

### Aesthetic Direction
- **Visual tone:** Dark mode only. Deep navy/black backgrounds with selective accent lighting.
- **References:** The reference UI shows a 3D Mapbox night map with glowing landmark buildings (ESB gold, WTC cyan), floating glass-morphism data cards with connector lines to map pins, and a structured AI response panel.
- **Anti-references:** No bright/flat Material Design. No rounded-pastel cards. No generic dashboard templates. Nothing that looks like a standard admin panel.
- **Theme:** Always dark. No light mode.

### Color System
- **Backgrounds:** #0c0c0f (base), #07111D (midnight), #111115 (surface)
- **Accents:** #15BFD2 (electric-cyan), #F2B35B (warm-amber/gold), #41E4F4 (highlight cyan)
- **Data-coded:** #84cc16 (health), #3b82f6 (permits/safety), #f59e0b (complaints/violations), #ef4444 (NYPD), #a855f7 (evictions), #06b6d4 (transit)
- **Glass:** bg-slate-950/40 backdrop-blur-2xl border-white/10

### Typography
- **Display:** Syne (Google Font) -- bold, uppercase, for titles and wordmarks
- **UI/Body:** DM Mono -- monospace for all labels, metadata, body text
- **Never use:** Inter, Roboto, Arial, or system fonts

### Design Principles
1. **Map-first:** The 3D map is the hero. All UI elements float above it as translucent overlays.
2. **Data has color:** Every data category has a fixed accent color. Colors encode meaning, not decoration.
3. **Glass over solid:** Panels use glass-morphism (backdrop-blur + low-opacity bg + thin borders), never opaque backgrounds.
4. **Animate with purpose:** Entry animations (slide-in, fade-in, scale) on data arrival. No gratuitous motion. Waveforms respond to agent state.
5. **Monospace precision:** UI text in DM Mono creates a technical, grid-aligned aesthetic. Display text in Syne adds cinematic weight.
